# Lesson: Introduction to APIs with Python

Welcome to the **Introduction to APIs** lesson! This repository covers the fundamentals of working with Application Programming Interfaces (APIs) using Python's powerful `requests` library. 

Through these hands-on examples, you will learn how to send `GET` requests, handle JSON responses, pass query parameters, and parse nested data structures (dictionaries and lists).

---

## 🚀 Learning Objectives
* Understand how to fetch data from public APIs using `requests.get()`.
* Convert API responses into Python dictionaries/lists using `.json()`.
* Navigate complex, nested JSON structures to extract specific data point elements.
* Pass query parameters to customize API outputs (e.g., location, dates).

---
